Question-1: What are the different ways to select an element in the DOM?

You can select elements using method like getElementById(), getElementsByClassName() and querySelector() for single elements or querySelectorAll() for multiple elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText and textContent is innerHTML set HTMl content and innerText get or set text and textContent get or set text ignoring style and HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation involves attaching a single event listener to parent element to handle events for child elements by capturing or bubbling the event.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event triggers on a child element and then propagates up through its ancestors to the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

To Create, add, remove element I use document.createElement(), appendChild(), remove() methods.
